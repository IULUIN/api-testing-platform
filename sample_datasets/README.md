# 示例数据集使用说明

## 文件清单

| 文件名 | 对应示例 | 变量 | 数据行数 |
|--------|----------|------|----------|
| 示例1_查询用户信息.csv | 示例1：查询用户信息 | user_id | 10行 |
| 示例2_查询文章列表.csv | 示例2：查询文章列表 | user_id | 5行 |
| 示例3_创建文章.csv | 示例3：创建文章 | title, content, user_id | 8行 |

---

## 使用步骤

### 第1步：重置数据库（如果需要）
```
双击运行：scripts\resetDb.bat
```
重置后示例测试用例会带上变量占位符，才能配合数据集使用。

### 第2步：上传数据集
1. 打开网页，点击导航栏「数据集管理」
2. 点击「上传数据集」
3. 选择本文件夹中的 CSV 文件上传
4. 三个文件分别上传

### 第3步：运行批量测试
1. 点击导航栏「测试用例」
2. 找到对应的示例（如示例1）
3. 点击「批量测试」按钮
4. 选择对应的数据集（如「示例1_查询用户信息」）
5. 点击「开始测试」

### 第4步：查看性能报告
1. 批量测试完成后，点击结果页右上角「📊 查看性能报告」
2. 或者点击导航栏「性能报告」查看所有历史报告

## 变量说明

### 示例1 - 查询用户信息
- URL：`https://jsonplaceholder.typicode.com/users/{{user_id}}`
- 变量 `{{user_id}}`：用户ID，取值 1-10
- 每行数据会请求不同用户的信息

### 示例2 - 查询文章列表
- URL：`https://jsonplaceholder.typicode.com/posts?userId={{user_id}}`
- 变量 `{{user_id}}`：用户ID，取值 1-5
- 每行数据会查询不同用户发布的文章列表

### 示例3 - 创建文章
- URL：`https://jsonplaceholder.typicode.com/posts`
- Body：`{"title": "{{title}}", "body": "{{content}}", "userId": {{user_id}}}`
- 变量 `{{title}}`：文章标题
- 变量 `{{content}}`：文章内容
- 变量 `{{user_id}}`：发布用户ID
- 每行数据会创建一篇不同的文章

---

## 注意事项

- 这些示例使用的是公开测试 API（jsonplaceholder.typicode.com），需要联网
- 示例3 的 POST 请求实际上不会真正创建数据，API 会返回 201 但数据不持久化
- 如果网络不通，测试结果会显示连接失败
