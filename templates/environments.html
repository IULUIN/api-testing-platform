💡 使用说明

第1步：添加环境，配置基础URL和环境变量

第2步：在测试用例中使用变量，如 {{"{{"}}base_url{{"}}"}}/api/users

第3步：执行测试时选择环境，系统会自动替换变量

第4步：设置默认环境，执行时自动使用

📝 配置示例

示例1：开发环境

名称：开发环境

基础URL：http://localhost:5000

环境变量：

{
  "base_url": "http://localhost:5000",
  "api_key": "dev_key_123",
  "timeout": "30"
}

示例2：测试环境

名称：测试环境

基础URL：http://192.168.1.100:8080

环境变量：

{
  "base_url": "http://192.168.1.100:8080",
  "api_key": "test_key_456",
  "timeout": "60"
}
